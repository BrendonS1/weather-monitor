================================================================================
                           WHY IT'S SET UP THIS WAY
================================================================================

The PostgreSQL database is collecting historical data so you COULD add:
  - Historical weather graphs (temperature over time, wind trends, etc.)
  - 24-hour history charts
  - Weather pattern analysis
  - Downloadable historical data

But right now, your frontend only shows LIVE data from the weather station.


================================================================================
                              TO ADD HISTORY CHARTS
================================================================================

You would need to create:
1. New API endpoint in proxy-server.js (or separate Python API)
2. Queries to PostgreSQL for historical data
3. New charts in index.html using the Plotly library (already loaded!)
4. Update frontend to fetch both live + historical data
