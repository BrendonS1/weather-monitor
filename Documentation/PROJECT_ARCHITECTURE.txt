
# WEATHER MONITOR PROJECT ARCHITECTURE
================================================================================
Last updated: 2026-02-15

================================================================================
                              SYSTEM OVERVIEW
================================================================================

Real-time weather dashboard for North Shore Airport (NZNE) with historical
data collection and wind history charts.


================================================================================
                           RENDER SERVICES (5)
================================================================================

  Service Name                        Type              File
  ──────────────────────────────────  ────────────────  ──────────────────────
  weather-monitor-frontend            Static Site       index.html
  weather-monitor-proxy               Web Service       proxy-server.js
  weather-monitor-worker              Background Worker weather_monitor.py

  weather-monitor-wind-api-service    Web Service       query_page/wind-api.js

================================================================================
                        API ENDPOINTS REFERENCE
================================================================================

CORS PROXY
  GET  https://weather-monitor-proxy.onrender.com/?url=[TARGET]
  Whitelisted targets:
    - http://weather.nsac.co.nz/NEmetData.txt
    - http://weather.nsac.co.nz/awibexport.txt
    - http://weather.nsac.co.nz/cams/CamInfo.txt
    - http://weather.nsac.co.nz/cams/*.jpg|*.png

WIND HISTORY API (public, no auth)
  GET  https://weather-monitor-wind-api-service.onrender.com/api/wind-history
  Returns: { timestamps[], wind_avg[], wind_xwind[] }
  Query:   Last 60 minutes from weather_update (filtered by device_utc_ts)

QUERY SERVER (password protected)
  POST https://query-weather-db-service.onrender.com/query
  Body: { password, sql }
  Returns: { columns[], rows[][] }
  Restriction: SELECT queries only
